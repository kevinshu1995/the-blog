# Blog

寫一些有的沒的

## 開發

```
# 開發模式
npm run dev

# 建置網站
npm run build

# 預覽建置結果
npm run preview
```

## 文章撰寫

在 `/posts` 目錄下建立 Markdown 文件，添加適當的 frontmatter 信息：

```
---
title: 文章標題 (必須要有)
date: YYYY-MM-DD
description: 文章簡短描述
tags:
    - 標籤1
    - 標籤2
category: 分類名稱(只能有一個分類)
---

文章內容...
```
